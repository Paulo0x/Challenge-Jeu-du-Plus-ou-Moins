# Challenge PowerShell - Jeu du Plus ou Moins

## Description
Un jeu interactif en ligne de commande developpe en PowerShell dans le cadre d'une formation chez TechSecure.

## Fonctionnalites implementees
* **Modes de jeu** : Solo contre l'IA ou Duel (2 joueurs).
* **Niveaux de difficulte** : Facile, Moyen, Difficile avec limites de tentatives.
* **Validation des entrees** : Gestion des erreurs de saisie (lettres, nombres hors limites).
* **Systeme de scores** : Sauvegarde locale des records dans un fichier CSV.
* **Interface** : Affichage en couleurs pour une meilleure experience.

## Comment lancer le jeu
1. Clonez ce depot.
2. Ouvrez un terminal PowerShell dans le dossier du projet.
3. Executez la commande suivante :
